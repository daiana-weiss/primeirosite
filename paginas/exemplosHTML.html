Exemplos de Tabelas

Tabela Comum

linha 1, coluna 1 linha 1, coluna 2
linha 2, coluna 1 linha 2, coluna 2
Como fazer o código:
<table border="1">
 <tr>
  <td>linha1, coluna 1</td>
  <td>linha1, coluna 2</td>
 </tr>
 <tr>
  <td>linha2, coluna 1</td>
  <td>linha2, coluna 2</td>
 </tr>
</table>

Tabela com 2 linhas de dados, 1 linha de cabeçalho e 2 colunas

cabeçalho 1 cabeçalho 2
linha 1, coluna 1
 linha 1, coluna 2
linha 2, coluna 1 linha 2, coluna 2
Como fazer o código:
<table border=1 width=100%>
 <tr>
  <th>cabeçalho 1</th>
  <th>cabeçalho 2</th>
 </tr>
 <tr>
  <td><center>linha1, coluna 1<center></td>
  <td align="right">linha1, coluna 2</td>
 </tr>
 <tr>
  <td>linha2, coluna 1</td>
  <td>linha2, coluna 2</td>
 </tr>
</table>

Tabela com exemplo de linhas de uma mesma coluna mescladas com rowspan

coluna 1 coluna 2 coluna 3
linha 1, coluna 1 linha 1, coluna 2 linha 1, coluna 3
linha 2, coluna 1 linha 2, coluna 2
Como fazer o código:
<table border=1>
 <tr>
  <th>coluna 1</th>
  <th>coluna 2</th>
  <th>coluna 3</th>
 </tr>
 <tr>
  <td rowspan="2">linha1, coluna 1</td>
  <td>linha1, coluna 2</td>
  <td>linha1, coluna 3</td>
 </tr>
 <tr>
  <td>linha2, coluna 1</td>
  <td>linha2, coluna 2</td>
 </tr>
</table>

Tabela com exemplo de linhas de colunas diferentes mescladas usando colspan

coluna 1 coluna 2 coluna 3
linha 1, coluna 1 linha 1, coluna 2 linha 1, coluna 3
linha 2, coluna 1 linha 2, coluna 2
Como fazer o código:
<table border=1>
 <tr>
  <th>coluna 1</th>
  <th>coluna 2</th>
  <th>coluna 3</th>
 </tr>
 <tr>
  <td>linha1, coluna 1</td>
  <td>linha1, coluna 2</td>
  <td>linha1, coluna 3</td>
 </tr>
 <tr>
  <td colspan="2">linha2, coluna 1</td>
  <td>linha2, coluna 2</td>
 </tr>
</table>

Exemplo de sua utilização:

Nota Fria Papelaria
quantidade descrição preço unitário preço total
2 lápis de cor R$5,00 R$10,00
4 caderno brochura 60fl R$3,00 R$12,00
Total R$22,00


Como fazer o código:
<table width=70% border=1 cellspacing="0" cellpadding="0" bgcolor="red" solid>
 <thead>
  <tr>
   <th>descrição</th>
   <th>preço unitário</th>
   <th>preço total</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <td>2</td>
   <td>lápis de cor</td>
   <td>R$5,00</td>
   <td>R$10,00</td>
  </tr>
  <tr>
   <td>4</td>
   <td>caderno brochura 60fl</td>
   <td>R$3,00</td>
   <td>R$12,00</td>
  </tr>
 </tbody>
 <tfoot>
  <tr>
   <td colspan="2">Total</td>
   <td>R$22,00</td>
  </tr>
 </tfoot>
</table>
